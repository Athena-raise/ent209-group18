# ent209-group18 - Tracking Health App (Huawei Health Style)

## App Introduction
This is a lightweight health tracking application inspired by Huawei Health. It focuses on stable, easy-to-use core health monitoring functions to help users visualize daily activities, sleep quality, and personal health status.

---

## Project Status
🟢 **ON TRACK**

## Key Decision
**Decision**  
Finalized 3 Must-have features for the health tracking app:
- Step counting & daily activity dashboard
- Sleep tracking & visualization
- Basic health data statistics & report

**Why**  
Limited development time and team resources; Huawei Health’s core value lies in basic tracking stability. Prioritizing 3 core functions ensures high-quality delivery.

**Alternative**  
Expanding to 5+ functions (heart rate, diet, AI advice). Rejected to avoid over-scoping and low delivery quality.
