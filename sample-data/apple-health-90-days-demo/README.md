# Apple Health 90-Day Demo Data

This folder contains synthetic Apple Health data for student project demos.

- `apple-health-90-days-demo.zip`: import this file from the app's Apple Health import button.
- `export.xml`: the same data as plain XML.

Date range: 2026-02-08 to 2026-05-08.

Parsed by the app with the `All` import window:

- 90 step days
- 89 sleep records
- 90 weight records
- 46 workout records
- 315 total imported records

The data is fictional and should only be used for demonstrations or testing.
